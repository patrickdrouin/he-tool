# Format:
#   - Top-level lines with no indent and no colon  →  group header (optgroup)
#   - Top-level lines with a colon                 →  ungrouped item  (id: Label)

#   - Lines starting with # and blank lines are ignored.
#
# The id is stored in the database; changing an id requires a migration.
# The label is only used for display and TSV export — safe to change freely.
